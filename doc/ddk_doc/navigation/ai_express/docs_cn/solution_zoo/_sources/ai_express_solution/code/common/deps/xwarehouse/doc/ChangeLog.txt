XwareHouse-2.0.6 [2019-1-9]
· Change：添加人种策略，对应不同的人种，使用不同的相似度计算。
· Change：接口变化，HobotXWHDataSourceInfo和HobotXWHSetInfo里添加is_check_attr_，是否校验属性，目前只校验人种。
· Notice：必须使用相似度来测试识别率指标，distance不能作为参考。
· Notice：如果入库有外国人的场景，is_check_attr必须设为1没有外国人场景，is_check_attr可设为0。
· Notice：当is_check_attr=1时，distance阈值和相似度计算参数设置不生效。
· Notice：is_check_attr = 1(校验人种) 的时候，底库大小只能是128。
