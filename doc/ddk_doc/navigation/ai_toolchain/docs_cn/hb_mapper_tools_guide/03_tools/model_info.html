2.4. hb_model_info 工具

hb_model_info 是用于解析混合模型(*.bin)编译时的依赖及参数信息的工具。

2.4.1. 使用方法

hb_model_info ${model_file}

2.4.2. 命令行参数

hb_model_info的命令行参数
--version

显示版本并退出。

-m

后接模型名称。当指定BIN_FILE为pack模型时, 仅输出指定模型的模型编译信息。

--help

显示帮助信息。
