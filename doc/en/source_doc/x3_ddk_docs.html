Appendix: Development Manual¶

Note

Please contact Horizon Robotics’ technical or sales personnel if you find any of below documentation links become invalid.

1. AI Toolchain Documentations¶

NO.

NAME

FILE LOCATION

DESCRIPTIONS

1

Horizon AI Toolchain User Guide

horizon_ai_toolchain_user_guide

The User Guide gives an overall picture of the AI Toolchain and helps understand the development process including development environment preparation, model deployment, application development, custom OP development etc. In addition, it also displays the Benchmark of multiple algorithm models.

2

HB Mapper Tools Guide

hb_mapper_tools_guide

Elaborates model conversion process and explains how to use related horizon_tc_ui tools.

3

Model Convert Sample Doc

hb_mapper_sample_doc

Introduces the horizon_model_convert_sample package, in which contains a series of samples that converting floating-point models into fixed-point models using the horizon_tc_ui, single image inference and accuracy eval samples.

4

Supported OP List and Restrictions

supported_op_list_and_restrictions

Elaborates Horizon’s platform supported Caffe and ONNX operators and restrictions.

5

Model Encryption

model_encryption

Explains how to use Horizon’s model encryption feature.

6

BPU SDK API DOC

bpu_sdk_api_doc

Introduces the API, data, struct, layout and alignment rules of Horizon’s runtime SDK.

7

Basic Sample User Guide

basic_sample

Explains a series of basic samples in the horizon_runtime_sample embedded development sample package. The package contains samples of single image inference, implementation and inference of models containing custom operators and batch processing of multi-models etc., helps developers accomplish fast model deployment

8

AI Benchmark User Guide

ai_benchmark

Explains a series of embedded application development samples in the ai_benchmark package. Developers can experience model deployment and model performance evaluation related application development. Sample scripts consist of performance and accuracy evaluations.
